# Kana Master

A Japanese kana learning app with spaced repetition.

## Features

- **SM-2 Algorithm** - Spaced repetition for efficient memorization
- **4 Study Modes** - Flashcard, Quiz, Writing, Pairing
- **Paired Display** - Shows hiragana and katakana together
- **Bilingual** - Chinese and English support
- **Dark Mode** - Light and dark theme

## Tech Stack

React + TypeScript + Vite

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
